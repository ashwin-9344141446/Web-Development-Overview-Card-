Full Stack Technology Cards – HTML & CSS Project;

Project Overview;
This project is a simple and professional web layout that presents an overview of important Full Stack Development technologies using structured content cards. The design is built using HTML for content structure and CSS Grid for layout styling.  
The goal of this project is to provide a clear, visually organized introduction to core full stack technologies including HTML, CSS, JavaScript, .NET, and MS-SQL.

Technologies Used;
HTML5 – Structure and content layout
CSS3 – Styling and grid layout design
CSS Grid – Page structure and alignment

Features;
Clean grid-based layout
Technology-focused information cards
Professional and readable content
Simple and beginner-friendly structure
Responsive design concepts applied
Styled buttons with hover effects

Technologies Explained in the Project;
HTML – Defines the structure of web pages  
CSS – Controls styling and layout  
JavaScript – Adds dynamic interactivity  
.NET – Backend application development framework  
MS-SQL – Database management system

How to Run the Project;
1. Download or clone the repository
2. Open the project folder
3. Double click `index.html`  
4. View the project directly in your web browser
